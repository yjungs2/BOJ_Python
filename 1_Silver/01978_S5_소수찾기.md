## 백준 > Silver5 > 1978번 : 소수 찾기
메모리 30840kb 시간 68ms  
https://www.acmicpc.net/problem/1978  

코드
```python
n = int(input()) # 주어진 개수 n
num = list(map(int, input().split(' '))) # 주어진 n개의 수 리스트
ans = n

for j in num:
    for k in range(2, j):
        if j % k == 0: # 소수가 아니라면
            ans -= 1 # 전체 개수 - 1
            break
if 1 in num: # 1은 소수가 아님
    ans -= 1
print(ans) # 소수의 개수
```
